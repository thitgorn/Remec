Touch You - A flexible key remote controller

Touch You

Introduction

TouchYou Application is the Android remote controller application that can adjust everything you want - Key , Icon , Size , Position. The best thing is that You can record the key and make it as the shortcuts for your work. For example , copying : ctrl+c : in one button in the mobile phone. Imagine, If you are using Adobe Photoshop , they have something like Control + Alt + T to Free transform the image. You don't want to remember that key so you Put it into TouchYou Application and save as some profile ! !.

System Requirement

Java version 8 or higher in the computer. Android version 6 or higher in the mobile phone.

How To Install
